# ESP32 嵌入式开发工程
## 一、项目介绍
本项目基于乐鑫ESP-IDF开发，实现了（比如：WiFi连接、串口数据收发、传感器数据采集）核心功能，用于模拟公司标准化嵌入式开发流程。

## 二、环境要求
- 主控芯片：ESP32-WROOM-32（改成你自己用的芯片型号）
- ESP-IDF版本：v5.2.1（改成你自己用的版本，必须写清楚，不然别人编译不了）
- 编译系统：Windows 10 / Ubuntu 22.04 / WSL2

## 三、目录结构说明
├── main/ # 核心代码目录，主函数 app_main、业务逻辑都在这里
├── config/ # 工程配置文件（分区表、SDK 默认配置）
├── docs/ # 项目说明文档、开发笔记
├── CMakeLists.txt # 顶层编译配置文件
├── .gitignore # Git 忽略文件配置
└── README.md # 项目说明文档（本文件）

## 四、编译&烧录步骤
1.  先配置好ESP-IDF开发环境，打开ESP-IDF终端（Windows）或执行环境变量脚本（Linux）
2.  设置芯片型号：`idf.py set-target esp32`（根据你的芯片型号修改）
3.  编译工程：`idf.py build`
4.  烧录+打开串口监控：`idf.py -p COM3 flash monitor`（COM3改成你自己的串口号）

## 六、开发规范
- 代码提交格式：`动作: 具体做了什么`，比如`feat: 新增WiFi连接功能`
- 所有临时编译文件不上传Git，统一在.gitignore里屏蔽
- 核心功能代码统一放在main目录，配置文件放在config目录
